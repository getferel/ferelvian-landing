# Ferelvian Landing Page

Modern, minimalist landing page for Ferelvian - Global Tech Product House.

## Products

- **Ferel** - News Intelligence Platform (ferel.com)
- **FinalForm** - Retail Intelligence Platform (finalform.club)
- **Workfield** - CRM + HRMS Platform (workfield.com)

## Deploy

### Option 1: Run Deploy Script

```bash
./deploy.sh
```

### Option 2: Manual Commands

```bash
# Add remote
git remote add origin https://github.com/anandhukrishna/ferelvian-landing.git

# Push to GitHub
git push -u origin main

# Deploy to Vercel
vercel --prod
```
